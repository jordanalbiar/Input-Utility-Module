# Input Utility Module

A fun, powerful userscript that makes typing on the web *way* more enjoyable.

Real-time font transformations, smart emoji & GIF triggers, **and a full built-in GIF Portal** — all in a clean, draggable floating panel.

## ✨ What’s New in v2.0

- **🌀 GIF Portal** — Search Tenor, GIPHY, and more directly inside the menu. Browse trending, search, save favorites, and insert GIFs with one click.
- Much better support for contenteditable fields (Google Docs, Notion, rich text editors, etc.)
- Improved GIF trigger management with **categories/tags**
- Overall polish and stability improvements

## Features

- **🔤 Fonts** — Real-time transformation into cool styles (Fraktur, Bubbles, Full-Width, Ancient, Script, Emoji Flags, etc.)
- **😀 Emoji Triggers** — Type `lol`, `:thinking:`, `<3`, `XD`, `:dog:`, etc. and they convert automatically
- **🎞 GIF Triggers** — Set up custom triggers that insert GIF URLs
- **🌀 GIF Portal** — Full GIF search engine built in:
  - Search Tenor & GIPHY (plus stickers)
  - Save favorites
  - One-click insert into any text field
  - Custom API sources supported
- **🎨 Themes** — Beautiful presets (Ocean, Mystic, Sakura, Fire, etc.) + full custom color editor
- **Custom Rules** — Add/edit your own emoji and GIF triggers
- **Import / Export** — Backup and restore your entire setup

## Installation

1. Install **Tampermonkey** (or Violentmonkey) in your browser.
2. Copy and paste the code into a new script on **Tampermonkey**.

Or manually:
- Create a new script in Tampermonkey
- Paste the entire script and save

## How to Use

1. Click any text field or textarea.
2. A small floating button appears near the bottom-right of the field.
3. Click it to open the control panel.

### Main Tabs:
- **🔤 Fonts** — Enable a font or create your own
- **😀 Emoji** — Manage emoji auto-replacements
- **🎞 GIFs** — Manage your custom GIF triggers + categories
- **🌀 Portal** — Search and browse thousands of GIFs (new!)
- **⚙️ Settings** — Adjust delays, themes, API keys, etc.
- **Export / Import** — Backup your config

The panel is fully draggable and resizable.

## Tips

- Font replacement is instant by default (change in Settings if you prefer a delay).
- GIF Portal works best with a free GIPHY API key (optional — Tenor works without one).
- You can create custom categories for your GIF triggers to stay organized.
- Works on most websites including rich text editors.
