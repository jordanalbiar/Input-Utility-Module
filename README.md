# Input-Utility-Module
A fun little userscript that makes typing on the web way more enjoyable, giving you real time font transformations, smart emoji and GIF triggers, all in a clean floating panel.

## What it does

- **Fonts**: Turn your text into cool styles like Fraktur, Bubbles, Full-Width, Ancient, Script, Emoji Flags, etc. Just pick one and start typing.
- **Emoji Triggers**: Type things like `lol`, `:thinking:`, `<3`, `XD`, `:dog:`, etc. and they turn into real emojis after a short delay.
- **GIF Triggers**: Type `:wave:`, `:facepalm:`, `:mindblown:` and get actual GIFs inserted.
- **Themes**: Comes with several nice presets (dark ocean, light mode, mystic, sakura, fire, etc.) plus full custom color control.
- **Custom Rules**: Add your own emoji or GIF triggers easily.
- **Import/Export**: Save and load your entire setup.

## Installation

1. Install **Tampermonkey** (or Violentmonkey) in your browser.
2. Create a new script in Tampermonkey via the extension button, and paste the Input Utility Module script code, ctrl + s to save.

## How to use

- Click the floating button (the one with "A" or a font emoji) to open the menu.
- **Fonts tab**: Pick a font or add your own.
- **Emoji / GIF tabs**: Add, edit, or disable triggers.
- **Settings**: Change delays, switch themes, tweak colors.
- **Export/Import**: Backup your config.

## Tips

- You can adjust the trigger delay if 1 second feels too long or short.
- The font replacement works instantly by default (you can add a delay in settings if you want).
- Custom fonts are added using simple character mapping — it's more powerful than it looks.
